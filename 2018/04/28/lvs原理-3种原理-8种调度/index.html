    集群

    集群就是多个独立的计算机通过网络组合成组合的计算机，共同参与任务，完成计算。

    特点

    高性能，性价比高，可伸缩，高可用

    集群架构分类

    1. 负载均衡集群（LBC）
    2. 高可用性集群（HAC）
    3. 高性能计算集群（HPC）

    负载均衡和高可用在互联网架构中经常使用

    –

    LVS负载均衡

    全称是linux virtual server ，是一个虚拟的服务器集群系统。

    三种工作模式

    1. VS/NAT模式
    2. VS/TUN模式
    3. DR模式

    调度算法

    1. 轮询算法（rr）
    2. 加权轮询调度（wrr）
    3. 加权最小连接数调度（wlc）
    4. 目的地址哈希调度（dh）
    5. 源地址哈希调度（sh）
    6. 最小连接数调度（lc）
    7. 基于地址的最小连接数调度（lblc）
